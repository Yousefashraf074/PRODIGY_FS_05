# Backend Architecture Overview

This backend will use Node.js, Express.js, TypeScript, PostgreSQL (Prisma ORM), Redis, JWT Auth, bcrypt, Multer, Cloudinary/S3, Socket.io.

## Key Folders

- src/
  - controllers/
  - routes/
  - models/
  - middlewares/
  - services/
  - utils/
  - config/
  - sockets/
  - prisma/
  - tests/

## Features

- Auth, user, post, comment, like, follow, notification, search APIs
- RESTful, secure, scalable, clean architecture
- Centralized error handling, logging, health checks
